Customize the draft

Here are some steps to get started. First, provide the teamnames. Then construct your preferred draft order. Now you should be good to go! Optionally, you can override the draft to include (or exclude) heroes and their images.

Construct the draft order

The draft order is constructed with JSON objects as shown below. The index indicates the draft position the side indicates which team and the type indicates a ban or a pick action.

      
         {"index": 1, "side": "1", "type": "ban"},

Override the heroes

To include a hero in the draft, just add a JSON object to the list with the hero's name.

      
         {"name": "adagio"}

If you want to override the default image, provide an img source. 

      
         {"name": "adagio", "img": "https://...adagio.png"}
